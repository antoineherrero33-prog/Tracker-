Protocole pré-run
Run court (L) : 1-2h avant → repas glucidique normal.
Fractionné (Me) ⚡ : 2h avant → 100-120g glucides. 30min avant → 30-40g rapides (banane, dattes).
Run long (V) : 2-3h avant → repas complet. Si >1h15 : 30-40g glucides toutes les 45min pendant l'effort.

4 métriques, rien de plus
Poids à jeun
Chaque matin
Calculer la moyenne sur 7 jours. C'est l'indicateur — pas le poids du jour.
Perf bench
Chaque séance A
Charge × reps. Stagnation 2 sem → ajuster programme, pas calories.
Tour de taille
1x/semaine
Au nombril, le matin. Baisse = perte de gras réelle, même si poids stable.
Calories
Chaque jour
Pèse 3 semaines pour calibrer ton œil. Ensuite l'œil suffit.
Règles d'ajustement
SI : Poids stagne 3 semaines
→ Réduire de 100 kcal sur les jours repos. Pas toucher aux jours muscu.
SI : Perte > 1 kg/semaine
→ Augmenter de 150 kcal sur jours run. Protéger la masse musculaire.
SI : Bench stagne 2 semaines
→ Vérifier sommeil et glucides pré-séance. Pas le programme.
SI : Fatigue excessive
→ Body Battery < 30 deux jours → transformer un run en repos actif.
