Protocole pré-run — Performance
Run court (Lundi) : 1-2h avant → repas normal glucidique (riz, flocons, pain). Pas besoin de rituel particulier.
Fractionné (Mercredi) ⚡ : 2h avant → repas riche en glucides (100-120g). 30 min avant → 30-40g glucides rapides (banane, dattes, pain blanc). C'est la séance qui fait progresser — elle doit être de qualité.
Run long (Vendredi) : 2-3h avant → repas complet. Si > 1h15 de course : 30-40g glucides toutes les 40-45 min pendant l'effort (gel, dattes, banane).
💡 La perf run s'améliorera aussi mécaniquement via la recompo : -5 kg de gras = environ -30 sec/km à même effort cardio.
Whey Isolate — Intégration
Jours muscu (Ma/J) : 1 scoop post-séance dans les 30-60 min. Priorité absolue.
Jours run : En collation si les protéines alimentaires n'atteignent pas 175g en fin de journée.
Jours repos : Optionnel. Miser sur sources alimentaires complètes (œufs, viande, fromage blanc 0%).

Poids à jeun

Charge × reps. Si tu stagnes 2 semaines → ajuster le programme, pas les calories.

1x/semaine

Calories
Chaque jour
Commence à peser 3 semaines pour calibrer ton œil. Ensuite l'œil suffit.
Règles d'ajustement
SI : Poids stagne 3 semaines
→ Réduire de 100 kcal sur les jours repos. Pas toucher aux jours muscu.
SI : Perte > 1 kg/semaine
→ Augmenter de 150 kcal sur jours run. Protéger la masse musculaire.
SI : Bench stagne 2 semaines
→ Vérifier le sommeil et les glucides pré-séance. Pas le programme.
SI : Fatigue excessive
→ Body Battery < 30 deux jours de suite → transformer un run en repos actif.
